I. Introduction

近年來，台灣車禍事故數量是年年上升，而每一次的事故，常常造成難以挽回的遺憾。為了盡可能避免車禍的發生，我希望能夠有一個模型能夠隨時去評估當時車輛以及外界狀況是否會造成死傷車禍的風險指數，所幸台中市警察局提供了近年各項車禍的詳細分析報告以供分析。

II. Data Collection

資料集來源: 政府資料開放平台
臺中市政府警察局110年1~10月份交通事故資料(目前最新只有到10月份的)

我所使用的資料集當中，包含了對於車禍現場狀況的詳細描述，如下圖:

我希望能夠做出一個模型，藉由輸入事故現場的詳細特徵，來預測是否會有死傷產生。因此，在我的模型當中，我的label會是死傷人數的部分，而feature則是資料中其餘部分

III. Preprocessing

Drop Unnecessary Columns

我去除掉的Columns包含:

  • 年、月、日: 因為我在進行Data Visualization的時候發現各日期車禍的死傷率並沒有太大的起伏
  • 分鐘: 因為分鐘尺度的時間對於車禍事件的影響並不大
  • 縣市: 因為都是在台中市
  • 車輛撞擊部位其他: 絕大部分的車禍資料這一項都是缺值，占了99%以上

Missing value

在這一部分，我原先是採用直接drop掉該筆資料的方式去實作，但是這樣子預測的準確率相當的低，始終無法突破70%以上。但當我這部分改成把NAN補0後，各模型的準確率都大幅上升，來到了95%左右。

Encoding

在encoding這一塊，我使用了one hot encoding，這是由於各資料欄位中的特徵值並沒有一個漸進的關係，因此不適合採用label encoding。

GPS經緯度

我將坐標軸的中心位置調整至經度120，緯度23.5。在完成這樣的調整之後我發現個模型的預測準確率上升了1.5倍左右。

Shuffle & Validation Method

我採用了Handout Validation with ratio 7:3。

IV. Models

SVM

經由前面HW3的結果，可以發現到使用rbf作為kernel的SVM效果較其他兩個linear以及ploynomial還要好。因此，在這項final project中我使用了rbf kernel的SVM。

svm = SVC(gamma='auto', kernel='rbf')

Random Forest

我使用了sklearn的RandomForestClassifier

RFC = RandomForestClassifier()

ANN

這邊我使用了2層隱藏層的神經網路進行實作，第一層大小為32，第二層為16。在每一個隱藏層之中，我使用了BatchNormalization以及Dropout來避免overfitting，並使用了ReLU作為激活函數避免gradient vanish問題。在輸出層這邊，我使用了sigmoid作為激活函數，輸出這次車禍的死傷機率。
在訓練的過程中，我設定了兩個回調函數，分別是EarlyStopping用來避免overfitting以及ReduceLROnPlateau用來適時減少學習率 我的loss function是使用BinaryCrossEntropy，並且搭配adam optimizer進行backpropogation

size_list = [32,16]
model = tf.keras.Sequential()
for i in size_list:
    model.add(layers.Dense(i))
    model.add(layers.BatchNormalization())
    model.add(layers.ReLU())
    model.add(layers.Dropout(0.5))
model.add(layers.Dense(1, activation='sigmoid'))

VI. Conclusion

這三個模型的訓練時間分別是:SVM 50秒，Random Forest 6秒，ANN 2分鐘。三個模型的準確率幾乎相同，但時間差異卻十分巨大。雖然現在是deep learning的時代，幾乎什麼東西用ANN去學習都可以有不錯的成果，但其需要長時間的訓練。不同類型的問題有其適合的模型，不一定甚麼都要用deep learning，傳統的maching learning在他們擅長的領域也有其優點。我這次選擇的問題剛好對於這三個模型都十分的適合，準確率都能達到99%以上，因此若是要為這項問題提出適合的模型，我應該會選擇速度較快的Random Forest。

藉由完成這個模型之後，我們可以透過在車內外裝設適當之感測器，實時收集各項數據並輸入到模型當中，並輸出造成死傷車禍的風險指數，供駕駛人參考。
